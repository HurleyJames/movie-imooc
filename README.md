# movie-imooc

<h6>利用豆瓣 API 编写的电影类微信小程序练手项目</h6>

## Override

* movie：包含当前豆瓣电影的页面，包括封面、评分等
* profile：用户的个人信息
* comment：用户评价页面暂时失效（因为豆瓣接口访问失效）
